LearningLock

Learn to earn screen time.

Kids earn banked minutes by using the learning apps you approve — then spend them to unlock the apps they love.

How it works

  • Set the exchange rate — e.g. 1 minute of reading earns 10 minutes of games.
  • Choose what counts as learning, and what gets unlocked: one app, a group, or everything.
  • Set daily caps so rewards stay balanced.
  • Built on Apple's Screen Time framework — activity data stays private on the device.
